How to play

Goal

The goal is to get the field cleared by removing all of the blocks from the field.

How to

To remove (better: To check whether a removal is valid) a block you have to click on it.
If you think you made a mistake, just undo your last action.

Rules

  1. After clicking on a colored block and the move is valid, all of the adjacent blocks of the same color are being removed
  2. Empty fields are being filled up with blocks from the top ('falling down')
  3. After 'falling down' the blocks shift to empty fields to the left side ('shift left')
  4. The game is over when there are no more blocks left or the player does not have any valid remaining moves

Game modes

  • Easy: Blocks are removed vertically, horizontally and diagonally
  • Normal: Blocks are removed vertically and horizontally
  • Harder: Select more colors and/or more fields to make the game harder
